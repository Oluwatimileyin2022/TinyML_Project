The Power of TinyML in Structural Analysis

The Background

A project about the use of TinyML for monitoring and detecting structural faults.This group project (GP) aims to develop and deploy a TinyML in an embedded device like raspberry pi to evaluate and monitor the condition of high-value assets but specifically the bolt looseness. High-value assets can be defined as any asset that is critical to a business.
